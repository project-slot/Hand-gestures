For this project I wanted to create a detection model focused on commonly used hand gestures, particularly those found in the emoji section on smartphones. Using Roboflow's tools, I collected and labeled a diverse dataset of hand gesture images, including gestures like thumbs up, peace sign, OK sign, and fist bump.
